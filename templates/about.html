ABOUT US

Our Vision: Making Fitness Personal

Fitness Coach is an LLM-powered solution that uses your personal information and preferences to create a tailored and detailed weekly plan. With a security-first approach, we employ the latest DevSecOps techniques to secure your data and interactions.

Our Capstone Project represents a significant technical achievement, bringing together the expertise of Ivo Garais in backend development, Gareth Enright in frontend design, and Michael Morley in DevSecOps. This collaboration aimed to create a robust, secure, and user-friendly web application that meets modern security and design standards.

Ivo focused on developing a reliable backend infrastructure, ensuring efficient data handling and secure interactions between the server and the client. This included implementing secure authentication mechanisms, data validation, and error handling to protect against common vulnerabilities.

Gareth applied his skills in frontend development to design an intuitive and accessible user interface. His work emphasized creating a responsive design that offers a seamless experience across different devices, prioritizing user experience without compromising on performance.

Michael's role in DevSecOps was crucial in integrating security practices throughout the development lifecycle. He implemented stringent security measures, including encryption for data transmission, secure coding practices, and regular security audits, ensuring the application met high standards of data protection and compliance.

Together, we built a platform that not only functions efficiently but also safeguards user data and provides a smooth and engaging experience. This project allowed us to apply and refine our technical skills in a collaborative environment, addressing real-world challenges and delivering a product that we are proud of.
